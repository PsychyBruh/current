## Usage

You **cannot** deploy Current on any static hosting services!

## Self-Hosting
   
```bash
git clone https://github.com/psychybruh/current

cd current

sh setup.sh
```

## Running locally

```bash
git clone https://github.com/psychybruh/current

cd current

bun i

bun dev
```
